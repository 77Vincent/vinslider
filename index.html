Why this one?

  • No dependencies. totally independent
  • Highly customizable
  • Super lightweight
  • Native Javascript
  • Good performance
  • Responsive
  • Easy to use

How to use it

Embed these two files into your project

vinslider.js

vinslider.css

Create DOM

<div class="vinslider">
    <ul>
        <li>content</li>
        <li>content</li>
        <li>content</li>
        ...
    </ul>
</div>

Instantiate in Javascript

var slider = document.querySelector('.vinslider');

var config = { duration: 1800, mode: 'carousel', .... };

new Vinslider(slider, config);

About

Yes, there are already a lot of similar modules out there to accomplish the same functionality, I have used several ones but found that they are not the exact stuff that I want, they either depend on a JS library, or with a not easy-to-use API, or with limited functionality, or has a bad performance. So I decided to create a module myself which could be as much simple, powerful, friendly, versatile as it can be. To achieve these goals, I choose native Javascript, manipulating native DOM element, and give all transition tasks to CSS, as you can see, those are the very basic elements that form a web page. And this is what I understand a basic module should be like. Check out the github repository page for more information, if you like it, feel free to share it with more people.
