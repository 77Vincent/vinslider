Vinslider

Super lightweight and smart Javascript slider module.

Why this one?

  • Super lightweight
  • No dependencies
  • Customizable
  • Easy to setup
  • Smart

How to use it

npm install --save vinslider
import Vinslider from 'vinslider'
Documentations

About

The goal of Vinslider is to make a super lightweight, easy-to-setup and, mean while, powerful and smart slider module. It's written in pure ES6. no other dependencies are needed. It's aware of DOM changes and will clean everything it has changed up, which means it's safe to make async request to a Vinslider instance. Check out the github repository page for more details, if you like it, feel free to share it with more people.
